tutorplugin plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/irtazaakram/tutor-contrib-tutorplugin

Usage
-----

::

    tutor plugins enable tutorplugin
